# Mask-Stock
📕 위치별 마스크 재고 검색 앱 📕

### 📕 위치별 마스크 재고 검색 앱 📕
*백업용 URL을 마스크 재고 API로 활용하여 마스크 재고를 확인하는 앱을 Flutter MVVM패턴으로 구현*   

#### 사용될 기술
1. 마스크 재고 API를 사용할 수 있도록 HTTP 통신
2. 위치를 정보를 제공하는 기술
3. Provider를 활용한 상태 관리
4. 구글 지도 오픈 라이브러리를 활용한 기술
